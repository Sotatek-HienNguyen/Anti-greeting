# System Architecture

## Frontend
- React (Vite) for fast development and execution.
- React Router for client-side navigation.
- Vanilla CSS with Glassmorphism for modern UI.

## Integrations
- Dictionary API: Real-time word definitions and audio.
- Web Speech API: Native browser speech-to-text.
- YouGlish: Contextual YouTube video integration.
- String Similarity: Semantic scoring for writing practice.
