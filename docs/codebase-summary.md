# Codebase Summary

## Overview
Anti-greeting is a React-based web application for conversational English learning.

## Key Directories
- `src/components`: UI components (Flashcard, SpeechInput, VideoContext).
- `src/pages`: Main application views (Home, Lesson, Writing).
- `src/data`: Static content (Word lists, sentences).
- `src/services`: Business logic and API integration.
- `docs`: Detailed project documentation.
