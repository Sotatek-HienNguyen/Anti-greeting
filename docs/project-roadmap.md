# Project Roadmap

## Phase 1: Core Experience
- Vocabulary Flashcards.
- Writing Mode with Speech-to-Text.
- Basic Dictionary Integration.

## Phase 2: Personalization & Mastery (Active)
- **Anki SRS Engine**: SM-2 based spaced repetition for 3,000 words.
- **Lexical Atelier UI**: High-end 3-pane study environment.
- **User Progress Tracking**: Mastery visualization and streaks.
- Advanced semantic feedback for Writing Mode.

## Phase 3: Community & Scale
- Collaborative learning tools.
- Mobile application.
